If the receiver has their device’s AirDrop feature switched on, they should appear as an option when the sender chooses ‘AirDrop’. The sender must then select the receiver. The receiver should then tap ‘Accept’ to receive the sent file(s).

Alternatively, both the sender and receiver can use a cloud storage app such as Dropbox or Google Drive. The sender must first upload the file to their cloud storage app and then send the link to the receiver. The receiver can then access the shared file by clicking on the link.
